# Advanced-Pong-Game
No boring Pong Game anymore! This version will improve the game experience.

The openning screen includes 2 options:
1. Boring Game - this is the regular known pong game. It has two players, the left player which presented as purple rectangle and the right player which presented as cyan rectangle. The players can move in the vertical direction only. The users can control the left and right players with w,s and up,down keys respectively. Each player needs to move in order to stop the ball from passing to his side. The score is shown in the top of the screen. The first player gets 5 scores wins.
2. Featured Game - this is the advanced version of pong game. All of the description about the regular version is true also here. In addition, some kinds of upgrades will show up throughout the game. One at a time, another upgrade will show up. And when the ball will hit an upgrade, it will affect the game in some way.

The upgrades are separated into 3 different types:
1. Players size changer - if the ball will hit it, then one of the players size will be changed (does not known in advance which player). If it is a green circle with a plus sign on it then the player gets bigger, otherwise it is a red circle with a minuse sign on it and then the player gets smaller. There are minimal and maximal lengths so don't panik :).
2. Ball size changer - if the ball will hit it, then the ball radius will be changed. If it presented as green/red circle with white circle in it then the ball will get bigger/smaller. Also here, there are minimal and maximal sizes for the ball so don't panik :).
3. Ball velocity changer - presented as a blue circle with asterisk sign (*) on it. if the ball will hit it, then the velocity of the ball (direction and power) will be changed to some other random velocity (direction and power). So prepare to be surprised! here you can panik..


Note: If the ball passed to one of the sides, it will return to the center automaticlly and start moving in a random velocity (direction and power). In this case ,the upgrate will stay in the same positions as they were before. Of course, when the ball hit an upgrade, it will disappear from the screen.


In the ending screen will be the final score of the game and an anouncment of the winner!
